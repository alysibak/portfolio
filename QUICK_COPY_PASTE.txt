# Quick Copy-Paste Code for Remaining Files

## File 1: app/experience/ExperienceClient.js

Copy your original experience page code, then:
1. Remove lines 1-45 (the useTheme hook and imports)
2. Add these imports at top:
   ```
   import { useTheme } from "@/components/ThemeProvider";
   import Navigation from "@/components/Navigation";
   import Footer from "@/components/Footer";
   import { experience } from "@/lib/data";
   ```
3. Remove the Navigation JSX (nav element, ~lines 70-110)
4. Replace with: <Navigation />
5. Remove the Footer JSX (footer element, ~last 80 lines)
6. Replace with: <Footer />
7. Change export name to: export default function ExperienceClient()
8. Use the experience data from imports instead of defining it locally
